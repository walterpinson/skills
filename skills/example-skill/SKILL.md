# Example Skill

## When to Use
Load this skill when you need a reference template for the correct SKILL.md structure.

## Procedure
1. Copy this folder to `skills/<your-skill-name>/`.
2. Rename the folder so it matches the `name` field in the frontmatter.
3. Replace this body with your skill's procedures, references, and asset links.
4. Add supporting files (scripts, references, assets) alongside `SKILL.md` as needed.
5. Add a row to the root `README.md` Skills table.
