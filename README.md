# Skills Library

AI skills, agents, instructions, prompts, and skill kits crafted to make AI-augmented development more effective.

**Target platform**: GitHub Copilot (Agent Skills)  
**Future platform**: Claude Code

---

## Artifact Types

| Type | Folder | Description |
|------|--------|-------------|
| Skills | [`skills/`](./skills/) | Independently usable agent skills (`SKILL.md`) |
| Agents | [`agents/`](./agents/) | Independently usable custom agents (`.agent.md`) |
| Instructions | [`instructions/`](./instructions/) | Independently usable instruction files (`.instructions.md`) |
| Prompts | [`prompts/`](./prompts/) | Independently usable prompts (`.prompt.md`) |
| Skill Kits | [`kits/`](./kits/) | Self-contained bundles of the above, initiated via an orchestration prompt |
