# Example Skill

## When to Use
Load this skill when you need guidance on structuring a kit-bundled skill.

## Procedure
1. Replace this body with your skill's step-by-step procedures.
2. Add supporting files (scripts, references, template assets) alongside `SKILL.md`.
3. Update `kit.json` `files.skills` globs if you add a `template/` subfolder.
